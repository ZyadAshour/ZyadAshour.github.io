PM Studio

A program created to assist programmers and scientists through their project planning stage. It allows users to write their algorithms in an IDE and share them with other users that need assistance in a similar problem via a Marketplace. It also allows scheduling tasks and making simple team management. More information can be found on PM Studio's Github Repository

Maze Running Robot

An original algorithm I made that allows a maze runner robot to fully explore a maze and guarantees that no cell in the maze will be missed. After the robot explores each cell in the maze and maps all of them, the algorithm uses Depth First Search to find the shortest path to the center and solve the maze. More information can be found on Maze Runner's Github Repository

Pulsating Stars Visualizer

An online 3D visualizer for pulsating stars. The visualizer allows users to view how some stars from the real-life pulsate, which is how their brightness change over time. It also allows users to create and manipulate stars, where each property change affects other properties. Ex: changing tempreature changes the star's color. More information can be found on Pulsating Stars Visualizer's Github Repository

CSharp Machine Learning

A machine learning library designed to utilize the powerful computational capabilities of the C# programming language. It relies on implementing each machine learning model from scratch using the mathematical reasoning behind that model's functionality, which enhances the code optimization and allows faster model training and testing exceution time. More information can be found on CSharp Machine Learning's Github Repository

Climate Change Monitoring Station

Another project that utilizes the computational capabilities of C#. This project is composed of a hardware circuit, an Arduino board that collects data from sensors and sends them to the internet, and a program that analyzes this data and uses machine learning to make predictions about how climate change will change the Red Sea in the future. More information can be found on Climate Change Monitoring Station's Github Repository

Simple Sign Language Detector

A project designed to help those with hearing disorders to communicate. The project utilizes OpenCV and Hands libraries to determine the position of each joint and hand landmark. Afterwards, with multiple calculations for the euclidean distance between these landmarks, the program predicts what the displayed gesture means. More information can be found on Simple Hand Language Detectors's Github Repository

Improving the Evasive-PDFMal Dataset

This paper that discusses a newly-published ML dataset and outlines some of its flaws, then makes multiple approaches to solve these flaws and succeeds in achieving better accuracy than any previous papers on that dataset.

Utilizing Computer Vision to Detect Contrails

This paper utilizes novel, state-of-art approaches using visual transformers to detect whether sky-images contain contrails or not. It also reflects on old approaches and tests them in comparison to the new approaches.

A litereature review on Deep Learning and Neural Networks

This paper delves into the history being deep learning and neural networks, how they started, and how they evolved. It also briefly explains how neural networks work and the mathematics behind their functionality.

Monitoring the effects of Climate Change on the red sea via an automated station

This was my final capstone project. Our team made a station composed of an Arduino Board along with TDS, Temperature, and CO2 sensors. The station independently collects data and sends them online, which is later collected and analyzed by a program to see how climate change affects the Red Sea. The project predicts how the aforementioned parameters would be in the future, and outlines the possible consequences on the marine life

Enhancing Egypt's electroplating industry with an automated feedback system

This was my third capstone project. The project's goal is to enhance the electroplating industry by automating it and providing it with a reliable feedback system to avoid and solve any potential errors in the thickness of the plated material. The project utilizes many original hardware techniques, including retriving data from a digital caliper and automating the thickness measurment process. It succeeded in eliminating any errors produced by the plating process.

Contributing to Egypt's near water-crisis using an atmospheric water generator

This was my second capstone project. The project's goal is to provide a clean and reliable source of water to avoid the increase of the Egyptian water crisis in the future. The project is an atmospheric water generator that condenses water vapour in the air and liquifies that vapour to clean water. This water was then used in the agricultural sector, specifically for bean crops. The project's results were promising, with a high rate of water collection.
